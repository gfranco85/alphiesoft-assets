# AlphieSoft Assets

Repositorio de recursos estáticos utilizados por los diferentes proyectos de AlphieSoft.

## Contenido

```
.
├── logo.png
├── logo-dark.png
├── logo-white.png
├── favicon.ico
├── banner-alert.png
└── icons/
```

## Uso

Estos recursos son utilizados por:

- Grafana
- Prometheus / Alertmanager
- Dashboards
- Aplicaciones Web
- Landing Pages
- Documentación
- Presentaciones
- Correos HTML

## URLs públicas

Una vez habilitado GitHub Pages, los archivos estarán disponibles mediante:

```
https://gbdvfranco.github.io/alphiesoft-assets/
```

Ejemplos:

```
https://gbdvfranco.github.io/alphiesoft-assets/logo.png
https://gbdvfranco.github.io/alphiesoft-assets/logo-dark.png
```

## Objetivo

Centralizar los recursos gráficos públicos para evitar duplicación entre proyectos y facilitar su reutilización.
